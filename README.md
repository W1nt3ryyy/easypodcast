# EasyPodcasts

Современное веб-приложение для прослушивания RSS подкастов. Реализовано на стеке React (фронтенд) и Django (бэкенд API) поверх базы данных PostgreSQL. Развертывание полностью автоматизировано с использованием Docker и Apache обратного прокси-веб-сервера.
